# Giáo Xứ Xuân Hòa Website

Website for Giáo Xứ Xuân Hòa (Xuân Hòa Parish Church) - A Catholic church website in Vietnam.

## Features

- Responsive design
- Image gallery
- Contact information
- Service schedules
- History section
- Priest information

## Technologies Used

- HTML5
- CSS3
- JavaScript (jQuery)
- Bootstrap 4
- Owl Carousel
- Magnific Popup
- AOS (Animate On Scroll)

## Setup

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process required - it's a static website

## Structure

```
├── css/          # Stylesheets
├── js/           # JavaScript files
├── images/       # Image assets
├── fonts/        # Font files
└── index.html    # Main HTML file
```
